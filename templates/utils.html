Firefox, local (unsigned): stays installed only on ESR / Developer Edition / Nightly — set xpinstall.signatures.required to false in about:config, then open the .xpi (Ctrl+O). Release Firefox keeps only AMO-signed builds; a temporary add-on via about:debugging is removed at every restart. Saved settings (server URL, scene) survive reinstalls either way — the extension has a fixed id.
Chrome: unzip, then chrome://extensions → Developer mode → Load unpacked.
